Get the kit

To design with the Carbon Axure Kit, you must have the most recent version of Axure RP installed.

1. Install Axure RP

2. Get the widget library

Download the .rplib        file to your computer. Open Axure RP and click the Add Library

icon at the top of the Libraries pane. Locate and select the IBM Carbon 10.rplib     file.

IBM Carbon 10.rplib

+

.rplib

This Axure RP file already has the settings that are required for proper display of elements in generated HTML prototypes.

Start designing

To get started, create a new file from the template (e.g. open and save a copy). Use the widgets from the “IBM Carbon 10” library that you have loaded into Axure RP.

Support and updates

If you are brand new to Axure RP, they offer tutorials to get you started.

Read our articles to learn about the role and usage of the special Axure RP widget types (dynamic panels, repeaters etc.) in the library. It will help you to configure the components and maintain their designed behavior.

Replacing the library won’t update the widget styles in your existing  project files automatically. Follow the recommended update process to make sure that components are up to date in your designs.
