Releases

Follow this update process to make sure that components are up to date in your designs.

Download the .rplib        file to your computer, and replace the old one with it. The updated version will be available in the Libraries pane.

.rplib

Update existing documents

Styles already in use prevail in an Axure RP document when adding widgets from a library. The existing style in the document will be applied to a widget that is dragged from the new library.

We make small adjustments and bug fixes to the kit on a regular basis. When a new version of the library is available, you need to download the new file to replace the old one. Releases will be posted to the GitHub repo.

If you’re an IBMer, join the #carbon-axure-kit Slack channel to get notified about changes.

Replacing the library won’t update the widget styles in your existing  project files automatically, therefore you need to follow a manual process to update the styles.

Import the styles from the library file into project documents using the Import Wizard.
