About repeater widgets

The "repeater" widget is an advanced widget type, used to display repeating patterns of text and images. Repeaters are containers, like dynamic panels, but instead of containing widgets in several states which are then switched between, repeaters contain one set of widgets—called the "item"—which is then iterated several times. Each iteration of the item is different, and the differences are controlled by data loaded into the repeater's "dataset".

Repeaters in this library

The primary benefit of constructing components using a repeater widget is that it is easier to maintain a consistent design between repeating elements, changing only the data set behind a repeater. Repeaters also provide dynamic control over items in a set.

Editing components with repeaters

To edit data set of a repeater, select the repeater and open the Style pane, where the Data section contains the editable data set.

In each component, repeaters are named with a "rpt" prefix (e.g. "rpt data"), so that you can search for them easier when filtering a list of widgets (e.g. for selecting as a target of action, or in the Outline panel).
